Reading: HTML5 & CSS3 For the Real World Chapter 8 - CSS3 Transforms and Transitions

The CSS3 transform property lets us translate, rotate, scale, and/or skew any element on the page. The order of transform functions does matter, we need to be careful about the order.

The translate(x,y) function moves an element x from the left, and y from the top: transform: translate(45px, -45px);
To mimic translation we can use position: relative;, and top and left values.

The scale(x,y) function scales an elementbythe defined factors horizontally then vertically: transform: scale(1.5, 0.25);
We can also mimic the scale function by altering the width and height, or changing the font-size of an element but this can change the layout.

The rotate() function rotates an element around the point of origin by a specified angle value.

The skew(x,y) function specifies a skew along the x and y axes: transform: skew(15deg, 4deg);

We can control the origin from which our transforms are applied using the transform-origin property: transform-origin: 10% 10%; (top or center).

The transition-duration property sets how long the transition will take: transition-duration: 0.2s;

The transition-timing-function lets us control the pace of the transition in even more granular detail. It can be slower or faster graduality:
transition-timing-function: ease; (linear, ease-in, ease-out, or ease-in-out.)

We can combine all the values of the transition properties in a single declaration: transition: transform 0.2s ease-out 50ms;

We can use a keyframe to include the properties we want to animate, along with the animated values.

Animation properties: animation-name, animation-duration, animation-timing-function, animation-iteration-count, animation-direction, animation-delay, animation-fill-mode and animation-play-state.
Like the transition properties there is a shorthand for animation properties too: animation: 300ms ease-in alternate 5s backwards appearDisappear;
