Reading: JavaScript-Novice to Ninja Chapter: 11 Further Functions

The call() method can be used to set the value of THIS inside a function to an object that is provided as the first argument: functionName.call()
The apply() method works in the same way: functionName.apply()

If a function takes some time to compute a return value, we can save the result in a cache property: functionName.cache = functionName.cache || {};

An Immediately Invoked Function Expression – or IIFE (pronounced “iffy”) – is an anonymous function that, as the name suggests, is invoked as soon as it’s defined.
